<h2>

<p>Course: QMSS G4063 Spring 2015<br>
Lecture: MW 1:10pm-2:25pm at 313 Fayerweather<br>
Office Hours: W 2:30pm-3:30pm at IAB 270C </p>

<h2>
<a id="user-content-instructor" class="anchor" href="#instructor" aria-hidden="true"><span class="octicon octicon-link"></span></a>Instructor</h2>

Lecturer in the Department of Statistics<br>
Columbia University</p>

<h2>
<a id="user-content-learning-objectives" class="anchor" href="#learning-objectives" aria-hidden="true"><span class="octicon octicon-link"></span></a>Learning Objectives</h2>

<p>... and resources to help you get there  </p>

<ul class="task-list">
<li>
<a href="http://www.sealthreinhold.com/tuftes-rules/rule_four.php">Tufte's Rules</a>. Above all else, show the data.</li>
<li>Grammar of Graphics. Wilkinson's theory and Wickham's implementation.</li>
<li>Meet your computer

<ul class="task-list">
<li>command line</li>
<li>text editors</li>
<li>file paths </li>
</ul>
</li>
<li>Working with data in <code>R</code> 

<ul class="task-list">
<li><a href="http://ecohen4.github.io/data-viz/r/R-tutorial.html">Basic training</a></li>
<li>Data analysis with <strong>plyr</strong>
</li>
<li>Data visualization with <strong>ggplot</strong>
</li>
<li>Scripting, debugging and writing functions</li>
<li>Reproducible research and dynamic output with <a href="http://rmarkdown.rstudio.com/RMarkdownReferenceGuide.pdf">Rmarkdown</a>
</li>
</ul>
</li>
<li>Communicating &amp; sharing your results in the browser

<ul class="task-list">
<li>github.io</li>
<li>
<code>HTML</code>, <code>CSS</code>, <code>JavaScript</code>
</li>
<li>Interactive visualization with <code>D3</code> </li>
</ul>
</li>
<li>Version control and collaboration with <code>github</code>

<ul class="task-list">
<li><a href="http://readwrite.com/2013/09/30/understanding-github-a-journey-for-beginners-part-1">Don't get scared, get started!</a></li>
<li><a href="https://help.github.com/articles/set-up-git/">Github Help</a></li>
<li><a href="http://skli.se/2012/09/22/introduction-to-git/">Git intro for readers</a></li>
<li><a href="https://www.youtube.com/watch?v=LXoWxrTdXkM">Git intro for visual and auditory learners</a></li>
<li><a href="http://skli.se/2012/10/07/git-workflow-beginner/">Git workflow for beginners</a></li>
<li><a href="https://help.github.com/articles/fork-a-repo">Forking</a></li>
<li><a href="https://www.youtube.com/watch?v=ifAEho6BmH0&amp;list=PLg7s6cbtAD17uAwaZwiykDci_q3te3CTY">Sharing and Collaborating</a></li>
</ul>
</li>
</ul>

<h2>
<a id="user-content-detailed-syllabus" class="anchor" href="#detailed-syllabus" aria-hidden="true"><span class="octicon octicon-link"></span></a>Detailed Syllabus</h2>

<p><strong>Subject to change</strong>. Syllabus will be updated periodically throughout the semester based on progress assessments and student learning outcomes. </p>

<ul class="task-list">
<li>Jan. 21 (W): Welcome! Introduction, context and history of Data Viz. Review syllabus and course resources.</li>
<li>Jan. 26 (M): Anscombe's Quartet. Statistical graphics. Visual model diagnostics. </li>
<li>Jan. 28 (W): <code>R</code> tutorial.</li>
<li>Feb. 02 (M): Quiz 1 (Tufte and Wilkinson Ch. 1-3). From Wilkinson's grammar of graphics to implementation with Hadley's <strong>ggplot2</strong>.</li>
<li>Feb. 04 (W): Meet your computer. Git and <code>github</code> tutorial.</li>
<li>Feb. 09 (M): Guest Lecture Michael Malecki - <strong>Data Maps</strong>. Reading due today: <a href="http://www.ncbi.nlm.nih.gov/pmc/articles/PMC3173776/pdf/nihms322499.pdf">value-by-alpha maps</a> &amp; <a href="http://andywoodruff.com/blog/value-by-alpha-maps/">more on value-by-alpha maps</a>
</li>
<li>Feb. 11 (W): Guest Lecture Swupnil Sahai - <code>Tableau</code>.</li>
<li>Feb. 16 (M): Time Series: The Climate Hockey Stick (IPCC). WSJ and NYT Business Section graphics. <a href="http://www.gapminder.org/">gapminder</a> </li>
<li>Feb. 18 (W): Handling time series in <code>R</code> with <em>date()</em> and <em>POSIX()</em> objects. In-class assignment: Handling geophysical time series -- example from <a href="http://ecohen4.github.io/ECREEE/#visually-inspect-the-data-with-respect-to-time">wind energy analysis</a>.</li>
<li>Feb. 23 (M):

<ul class="task-list">
<li>Quiz 2! </li>
<li>Quiz review.</li>
<li>Project status updates. </li>
<li>Narrative Graphics of Space and Time: Minard's narrative of Napoleon's Failed Conquest of Russia. <a href="http://www.informationisbeautiful.net/visualizations/the-microbescope/">NYT Ebola Graphics</a>. Ornithology and botany field guides. </li>
<li>Reading due today: Tufte chapters 4-9. Wilkinson chapters 4-7 (you can skim ch. 5). </li>
<li>HW2 due Monday Mar. 2nd: Re-Create Minard's Napolean Conquest <em>or</em> Ebola/contagion graphic in <code>R</code>.</li>
</ul>
</li>
<li>Feb. 25 (W): 

<ul class="task-list">
<li>
<code>R</code> to the next level (review):

<ul class="task-list">
<li>Split-Apply-Combine with Hadley's <a href="http://www.jstatsoft.org/v40/i01/paperre">plyr</a>. </li>
<li>Reshape data with <a href="http://cran.r-project.org/web/packages/reshape2/index.html">reshape2</a>. </li>
<li>Create the expected data structure for <a href="http://docs.ggplot2.org/current/">ggplot</a>.</li>
</ul>
</li>
<li>
<code>git</code> to the next level (new): 

<ul class="task-list">
<li>Undo things with git <a href="https://www.atlassian.com/git/tutorials/undoing-changes/git-revert">revert</a> and git <a href="https://www.atlassian.com/git/tutorials/undoing-changes/git-reset">reset</a>
</li>
<li>Resolve merge conflicts with git <a href="http://git-scm.com/docs/git-mergetool">mergetool</a> and from the <a href="https://help.github.com/articles/resolving-a-merge-conflict-from-the-command-line/">command line</a> </li>
</ul>
</li>
</ul>
</li>
<li>Mar. 02 (M): HW2 due today. Project status updates due today. In class: Scripting, debugging and writing functions in <code>R</code>.</li>
<li>Mar. 04 (W): Peer code review of HW2 (30min). Time to work on projects (45min). Elliot at conference in Portugal; Hueiyi will be available for questions.</li>
<li>Mar. 09 (M): Colors! Semantically resonant colors. Color pallettes. HEX.</li>
<li>Mar. 11 (W): Mid-term Quiz</li>
<li>Mar. 16 (M): SPRING BREAK</li>
<li>Mar. 18 (W): SPRING BREAK</li>
<li>Mar. 23: Cartography with <strong>Data Maps</strong>, <strong>rMaps</strong>, <strong>Maps</strong> or other <code>R</code> libraries.</li>
<li>Mar. 25: TBD based on mid-term progress assessment. </li>

<li>FINAL EXAMS</li>
