This R code takes in the microbe data and recreates the microbe-scope graphic as well I could. In order to scale the y-axis unevenly as it is in the original graphic (log scale: 0-10, normal scale:10-100) I manually manipulated the data to replicate this scale. I split the data into three different data frames to make this manipulation easier and then binded them back together. Because log(0)=-inf, I set all case.fatality.rates that were 0 to .01 and Norovirus to .02 before scaling. This helped ot replicate them as 0 and .01 after the log transformation.

Notes:

  • I left the primary.mode.of.transmission for cholera blank because the information in the data set and original graphic were conflicting.
  • I decided to leave in the diseases that were omitted in the original graphic
  • The text size was made very small so that it looked good in the knitted HTML
setwd("~/Github/data-viz/data-viz/ebola")
library(ggplot2)
Test<-read.csv("microbe-scope-bugs.csv")

#manually remove entries that shouldn't be included in graphic
Test<-Test[-c(1,2,18),]

#drop percent and convert both variables to numeric
dropPct<-function(df, var){
  dropPct<-strsplit(as.character(df[[var]]), split="%")
  df[[var]]<-as.numeric(dropPct)
  return(df[[var]])
  }
Test[["case.fatality.rate"]]<-dropPct(Test, "case.fatality.rate")
Test[["average.basic.reproductive.rate"]]<-dropPct(Test, "average.basic.reproductive.rate")

#order data for easy subsetting
Test<-Test[order(Test$case.fatality.rate),]

#create subset of data with values ~=0 and do log transformation 
#+.01 because log(0)=-inf
zeroTest<-Test[-c(6:38),]
zeroTest$trans.fatality<-log10(zeroTest$case.fatality.rate+.01)*10

#create subset of data with .01 < values <10 and do log transformation
lowTest<-subset(Test, case.fatality.rate<10)
lowTest<-lowTest[-c(1:5),]
lowTest$trans.fatality<-log10(lowTest$case.fatality.rate)*10

#create subset of data > 10 and do not transform data
highTest<-subset(Test, case.fatality.rate>10)
highTest$trans.fatality<-highTest$case.fatality.rate

#bind zero, low, and high data frames together to create manually transformed data
sumTest<-rbind(zeroTest,lowTest,highTest)

#create scatter plot with manually transformed data and relabel y-axis to reflect log transformation
psum<- ggplot() + geom_point(data=sumTest, aes(x=average.basic.reproductive.rate, y=trans.fatality, colour=primary.mode.of.transmission)) + geom_text(data=sumTest, aes(x=average.basic.reproductive.rate, y=trans.fatality, colour=primary.mode.of.transmission, label=X), size=2, vjust=0, hjust=0)+scale_x_continuous(breaks=c(0:18)) + scale_y_continuous(breaks=seq(-20,100,10), labels=c(0,.1,1,seq(10,100,10)))+theme_bw()+labs(title= "The Microbe-Scope",x="CONTAGIOUSNESS", y="DEADLINESS")+coord_fixed(ratio=1/10)
psum
